# 📚 Library Management System

A full-stack Library Management Web Application built using Django that allows efficient management of books, students, and borrowing workflows through a modern dashboard interface.

## 🚀 Features

📖 Add, view, and manage book records

🔄 Book issue and return tracking

👤 Student login portal to view issued books

🔐 Secure authentication using Django’s user system

🔍 Search books by title or author

📊 Dashboard showing total, issued, and available books

🧑‍💼 Admin panel for managing books and students

🎨 Premium responsive UI with sidebar layout

## 🛠 Tech Stack

Backend: Python, Django

Database: SQLite

Frontend: HTML5, CSS3, Bootstrap

Authentication: Django Auth System

## 🧠 Architecture

This project follows Django’s MTV (Model–Template–View) architecture:

Models → Database structure (Books, Students)

Views → Business logic & request handling

Templates → Frontend pages

URLs → Routing

# 👥 User Roles
## Librarian (Admin)

Manage books

Track issue/return status

View dashboard statistics

## Student

Login securely

View their issued books
